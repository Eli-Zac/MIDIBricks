Hub
Multiple hubs stay in sync over Bluetooth — each plays one track. Available on Prime / Inventor hubs. The leader is the master clock and followers re-align to it continuously, so they won't drift over a long song. Each hub shows its track number on its screen. Sync trim (default −100) pulls followers forward to cancel BLE broadcast latency — make it more negative if followers lag, less if they run ahead. Countdown (default 3 s) runs after you press start: the leader counts down while every hub locks onto the shared clock, so they begin the first note already in sync — then all hubs show a "playing" light animation.
