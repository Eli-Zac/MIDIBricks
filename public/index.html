Hub
Multiple hubs stay in sync over Bluetooth — each plays one track. Available on Prime / Inventor hubs. The leader is the master clock and followers re-align to it continuously, so they won't drift over a long song. Each hub shows its track number on its screen. Sync trim (default −200) pulls followers forward to cancel BLE broadcast latency — make it more negative if followers lag, less if they run ahead.

Hub 1 is the leader: run the follower hubs first, then press the leader hub's center button to start them all together. Press it again to stop every hub.
